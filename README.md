# 🚀 Curso em C++ — Agência Espacial Brasileira (AEB)

Este repositório contém exercícios desenvolvidos durante um **minicurso da Agência Espacial Brasileira (AEB)**, com foco na introdução à programação em **C++** e aplicação de conceitos básicos em contextos práticos.

---

## 📌 Sobre o Projeto

O projeto foi desenvolvido como parte de um **minicurso educacional**, com o objetivo de praticar lógica de programação e fundamentos da linguagem **C++**.

Os exercícios evoluem progressivamente, começando com conceitos básicos e avançando até aplicações relacionadas a cálculos aeroespaciais.

---

## 🧠 Exercícios Desenvolvidos

### 🔹 Exercício 1 — Entrada e Exibição de Nome

Programa básico que:

- Solicita ao usuário que digite seu nome e sua idade
- Exibe o nome e idade digitado na tela
- Trabalha com entrada e saída de dados

---

### 🔹 Exercício 2 — Calculadora Simples

Programa que:

- Realiza operações matemáticas básicas
- Trabalha com operadores aritméticos
- Recebe valores digitados pelo usuário

---

### 🔹 Exercício 3 — Jogo de Adivinhação

Programa que:

- Gera um número aleatório
- Solicita tentativas do usuário
- Informa se o número está correto

---

### 🔹 Exercício 4 — Calculadora Aeroespacial 🚀

Programa voltado para cálculos relacionados ao voo, como:

- Distância atingida
- Distância percorrida
- Tempo de voo

Este exercício simula aplicações práticas relacionadas à área aeroespacial.

## ▶️ Como Executar o Projeto no VS Code

### 🛠️ Requisitos

Para executar este projeto, é necessário ter instalado:

- Visual Studio Code
- Compilador C++ (g++)
- Git

Siga os passos abaixo para executar os programas utilizando o Visual Studio Code.

### 📥 1. Clonar o repositório

Abra o terminal do VS Code e execute:

```bash
git clone https://github.com/seu-usuario/curso-c-AEB.git
```

Depois entre na pasta:

```bash
cd curso-c-AEB
```

### ⚙️ 2. Compilar o programa

Abra o terminal do VS Code e execute:

```bash
g++ src/ex03_jogo_adivinhacao.cpp -o programa
```
(Substitua pelo arquivo desejado.)

Exemplo:

```bash
g++ src/ex01_nome_usuario.cpp -o ex01
g++ src/ex02_calculadora.cpp -o ex02
g++ src/ex03_jogo_adivinhacao.cpp -o ex03
g++ src/ex04_calculo_aeroespacial.cpp -o ex04
```

### ▶️ 3. Executar o programa

Após compilar, execute:

```bash
./programa
```

Ou:

```bash
./ex01
./ex02
./ex03
./ex04
```
